# WhyPayFees — whypayfees.co.uk

Melbourne, Derbyshire's commission-free private property listings site.
Static HTML/CSS/JS, hosted on Cloudflare Pages, deployed from this GitHub repo.

- **START-HERE.md** — how to put this site online (15 minutes, browser only)
- **RUNNING-THE-SITE.md** — how to run it day-to-day with Claude as your back office
- `js/listings.js` — the one data file Claude edits to add/update/sell properties
- `templates/listing-template.html` — the page template Claude fills for each new home
